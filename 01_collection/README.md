# 01_collection (Raw data collection)

이 폴더는 **원천 데이터 수집 단계**를 정리한 곳입니다.

## 폴더 구성

- `code/`: 국가별/소스별 데이터 수집 코드(스크래핑, 공개 CSV 다운로드 등)
- `raw/`: 수집된 원천(raw) 데이터 저장 위치 (가공 전)
- `docs/`: 데이터 출처/수집 경로 정리(국가별 표 등)

## 실행 방법(예시)

> 실제 실행 파일명은 `code/`에 정리된 뒤 여기서 업데이트합니다.

- (예) 특정 국가 수집:
  - `python code/<script_name>.py`

## 데이터 출처

국가별 상세 출처/수집 방식은 `docs/data_sources.md`에 정리합니다.
